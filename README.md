## 📁 프로젝트 소개
부모와 아이들이 함께 추억을 쌓을 수 있는 문화 행사 정보를 제공하고, <br />
부모들이 육아 및 아이 용품 등에 관하여 서로 유익한 정보를 주고받으며 소통할 수 있는 공간을 마련해 준다.
